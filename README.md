# Partners Archive

Arquivo de botões de parceria 120×40.

## Estrutura

```
src/
  partners.json         ← arquivo de parceiros
public/
  buttons/              ← imagens dos botões
    habbo.gif
    habbotalk.gif
    ...
```

## Adicionar um parceiro
